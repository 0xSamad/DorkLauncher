# DuckDuckGo DorkLauncher

A simple professional GUI tool for launching dork-based search queries on DuckDuckGo using Python and Tkinter.

## Features
- GUI to load custom dork lists
- Target any domain
- Open results in browser batch-by-batch
- Sleek modern look
